# 快速開始

AgentKit 是 Inventai 打造的 Claude Code 工具市集，提供 Skill、Agent、MCP Server、Plugin 四種套件類型。

## 瀏覽套件

前往 [瀏覽頁面](/agentkit/browse) 即可依類型、標籤篩選所有套件。點擊套件卡片進入詳情頁，查看說明文件與安裝指令。

## 安裝套件

每個套件詳情頁都提供針對不同工具的安裝指令：

- **Claude Code** — 最完整支援
- **Cursor** — 僅支援 Skill
- **手動下載** — 適合任何編輯器

詳細安裝說明請參閱各類型頁面：[Skill](/?page=types/skill)、[Agent](/?page=types/agent)、[MCP Server](/?page=types/mcp)、[Plugin](/?page=types/plugin)。

## 發布套件

登入 GitHub 後，前往 [發布頁面](/agentkit/publish)。填寫套件資訊並貼上主體內容後提交。提交後會自動開一個 Pull Request 到對應的 category repo，經審核後合併即完成發布。

詳細流程請參閱 [發布套件](/agentkit/docs?page=publishing) 文件。
